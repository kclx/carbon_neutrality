# 个人碳排放量计算器

## 项目简介

这是一个基于香港中小企业碳审计工具箱数据开发的个人碳排放量计算器网页应用。帮助个人了解日常生活中的碳足迹，并提供针对性的减排建议。

## 功能特点

### 📊 全面的碳排放计算
根据以下8个维度计算个人碳排放：
1. **饮食排放** - 包括肉类、蔬菜、主食等
2. **通勤排放** - 工作日的通勤方式和距离
3. **周末出行排放** - 休闲娱乐的交通排放
4. **航空排放** - 短中长途航班的碳排放
5. **衣物排放** - 购买新衣物产生的碳排放
6. **家居用品排放** - 纸张、塑料等消耗品
7. **用水排放** - 洗澡和洗衣用水
8. **用电排放** - 根据不同电力公司计算

### 🔢 计算公式

```
总排放(周期) = (饮食排放 + 通勤排放 + 周末出行排放 + 航空排放
              + 衣物排放 + 家居用品排放 + 用水排放 + 用电排放)
              × 回收系数
```

**回收系数：**
- 积极回收：0.95 (减排5%)
- 偶尔回收：1.0 (无影响)
- 不回收：1.05 (增排5%)

### 🌳 碳抵消计算
自动计算需要种植多少棵树才能抵消产生的碳排放
- 基准：每棵5米高的树每年吸收 23kg CO₂

### 💡 智能建议
根据各项排放数据，提供针对性的减排建议

## 使用方法

### 方式一：直接打开HTML文件（推荐）

1. 找到文件 `personal_carbon_calculator.html`
2. 双击用浏览器打开
3. 填写调查问卷
4. 点击"计算我的碳足迹"按钮

### 方式二：使用本地服务器

```bash
# 在项目目录下运行
python -m http.server 8000

# 或者使用 Python 2
python -m SimpleHTTPServer 8000

# 然后在浏览器中访问
http://localhost:8000/personal_carbon_calculator.html
```

## 数据来源

所有碳排放系数基于以下权威来源：

1. **香港中小企业碳审计工具箱** (2010)
   - 香港大学
   - 香港城市大学
   - 香港特区政府机电工程署和环境保护署

2. **主要参考文献**
   - IPCC NGGIP 排放因子数据库 (2006)
   - PAS 2050:2008 规范
   - WRI/WBCSD 温室气体议定书

## 使用示例

### 典型香港上班族一个月的碳足迹

**基本信息：**
- 计算周期：一个月
- 回收习惯：偶尔回收

**饮食：**
- 猪肉：50克/天
- 鸡肉：50克/天
- 鱼：30克/天
- 蔬菜：300克/天
- 米饭：400克/天

**通勤：**
- 方式：地铁
- 距离：10公里单程
- 工作日：5天/周

**用电：**
- 300千瓦时/月
- 中华电力

**预计月碳排放：** 约 250-350 kg CO₂

## 减排建议

### 🥗 饮食方面
- 每周减少1-2餐肉类，可减排 15-20%
- 选择当地时令蔬菜
- 减少食物浪费

### 🚇 交通方面
- 优先公共交通（地铁 > 电车 > 巴士）
- 短距离选择步行或骑车
- 拼车出行

### ⚡ 用电方面
- 使用节能电器（一级能效）
- 及时关闭待机电源
- 夏季空调设置26°C以上

### ♻️ 其他建议
- 积极垃圾分类和回收
- 理性购物，减少不必要消费
- 选择可重复使用的物品
